# Excalidraw Artist

创建专业的 Excalidraw 手绘风格图表，输出 `.excalidraw` 文件。

## 支持的图表类型

| 类型 | 适用场景 |
|------|----------|
| 架构图 | 系统设计、微服务、分层架构 |
| 流程图 | 业务流程、审批流程、部署流程 |
| 泳道图 | 多角色协作、跨部门流程 |
| 时序图 | API 调用、消息流转 |
| 线框图 | UI 原型、页面布局 |

## 快速开始

```bash
# 用户请求示例
"画一个微服务架构图，包含网关、用户服务、订单服务"
"画个审批流程图"
"帮我画个登录时序图"
```

输出文件可在 [excalidraw.com](https://excalidraw.com) 打开编辑。

## 设计原则

### 布局
- 水平流程：从左到右
- 垂直层级：从上到下
- 间距一致：元素间 40-60px
- 整体宽度：1200-1600px

### 配色（推荐，可根据场景调整）

**商务风格**（默认）：
| 用途 | 颜色 |
|------|------|
| 主色 | #1e3a5f |
| 次色 | #4a90d9 |
| 背景 | #f1f5f9 |
| 强调 | #10b981 |
| 边框 | #6b7b8c |

**极简风格**：主色 #1f2937，边框 #9ca3af，背景 #ffffff

### 元素选择

| 类型 | 形状 | 用途 |
|------|------|------|
| 核心组件 | 圆角矩形 | 服务、模块 |
| 流程节点 | 矩形 | 步骤、操作 |
| 决策点 | 菱形 | 判断、分支 |
| 数据源 | 圆柱形 | 数据库 |
| 起止点 | 椭圆 | 开始、结束 |

### 连接线

| 样式 | 用途 |
|------|------|
| 实线箭头 | 主流程 |
| 虚线箭头 | 可选/异步 |
| 双向箭头 | 双向调用 |

## 工作流程

1. **理解需求** → 确定图表类型、识别关键元素
2. **规划布局** → 确定尺寸、划分区域
3. **构建元素** → 参考 `references/examples.md` 的 JSON 结构
4. **输出文件** → 生成 `.excalidraw` 文件

## 参考文档

| 文件 | 内容 |
|------|------|
| `references/examples.md` | 3 种图表的完整 JSON 示例 |
| `references/element-ref.md` | 元素属性速查表 |

## 注意事项

- 复杂图表建议分步骤描述需求
- 大型图表（>30 元素）建议拆分
- 生成后可在 Excalidraw 中微调
- 中英混合标注增强可读性
